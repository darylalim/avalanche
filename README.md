# Avalanche
Final project for the [Fast Prototyping of GenAI Apps with Streamlit](https://www.coursera.org/learn/fast-prototyping-of-genai-apps-with-streamlit) course by [DeepLearning.AI](https://www.deeplearning.ai/) and [Snowflake](https://www.snowflake.com/).

Understand customer feedback faster with a data processing application powered by generative AI.

Avalanche sells outdoor gear. The dataset includes reviews for winter products.

## Dependencies
- python-dotenv - Load Anthropic API key from an environment variable file.
- Pandas - Filter data.
- Anthropic API - Analyze sentiment in customer reviews with a large language model.
- Plotly - Plot average sentiment scores by product and delivery status with interactive data visualizations.
- Streamlit - Build a prototype application with a user interface and publish on Streamlit Cloud.
